# Changelog

## [0.3.0] - 2026-01-29

Initial public release of Helio.

### Features
- Natural language queries for PV simulations
- Multi-agent architecture for accurate results
- Support for multiple LLM models via OpenRouter
- Secure sandboxed code execution
- Cross-platform support (Windows, Linux, macOS)

### Simulation Capabilities
- Annual energy calculations
- Tilt optimization
- Tracking system analysis
- System loss calculations
- Monthly/daily energy profiles
